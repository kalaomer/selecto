Selecto

Selecto is a custom select element for select.

Exmaple



<select name="" class="demo-1" id="" placeholder="Select">
	<option value="">Lorem.</option>
	<option value="">Rerum!</option>
	<option value="">Autem!</option>
</select>

<script>
	$('.demo-selecto .demo-1').Selecto();
</script>

This is a way for convert select to Selecto.
